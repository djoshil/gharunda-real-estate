IMAGES FOLDER — REPLACE BEFORE GOING LIVE
==========================================

This folder contains placeholder gradient images so the site works
out of the box. Swap each one for a real, licensed photo with the
SAME FILE NAME (or update the path in index.html / css/style.css if
you rename them).

  hero-bg.jpg           -> Full-screen hero background
                           Recommended: 1800x1000px+, landscape,
                           luxury skyline / residential towers.

  about-building.jpg    -> "About Us" section image
                           Recommended: 1000x1250px (4:5), portrait,
                           modern residential building.

  luxury-villa.jpg      -> "Why Choose Us" section image
                           Recommended: 1000x1250px (4:5), portrait,
                           premium villa / bungalow.

Tip: keep every image under ~300KB (compressed JPG/WebP) so the
homepage stays fast to load.
